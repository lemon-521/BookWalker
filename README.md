# 书中的“灵魂摆渡虾”

> **让每一本书，都成为一场可以跨越时空的对话。**

---

## 📖 项目简介

**灵魂摆渡虾**是一个基于 OpenClaw 框架的沉浸式阅读对话智能体。它让读者可以与书中人物进行深度对话，将阅读从单向接收升级为双向交流，用科技对抗都市阅读孤独。

当你在读《三体》时，可以和**大史**聊聊宇宙社会学；当你在读《活着》时，可以听**福贵**讲他的一生；当你在读《小王子》时，可以和**小王子**一起看星星；当你在读《西游记》时，可以和**孙行者**论论道、降降妖。这不是简单的角色扮演，而是基于对文本的深度理解，让 AI 真正“进入”书中世界。这不是简单的角色扮演，而是基于对文本的深度理解，让 AI 真正“进入”书中世界。

## 🎯 解决什么问题

2025 年数据显示，我国成年人人均纸质书阅读量仅 4.79 本，**图书的完读率不足20%**，挖财记账和天猫图书的数据显示，国民人均读书消费增长，但"购入的书籍并非都被阅读完了，可谓'买书如山倒、读书如抽丝'"。**亚马逊Kindle数据**：2015年度报告显示，**超过一半的畅销书没有被读完**；销量前十的付费电子书中，只有4部进入阅读完成率前20名。

深层原因不是“没时间”，而是 **“读不进去、无人可聊、读完就忘”**——面对静态的文字，没有互动、没有反馈、没有共鸣，阅读的热情被孤独消磨殆尽。

**灵魂摆渡虾**要做的，就是让阅读从孤独的独白，变成一场跨越时空的对话，成为一场可以走进的人生故事。

## 🦞 这只“龙虾”能干什么

### 🎭 化身书中人物
- 读《三体》时，AI 化身为**大史**——说话直来直去，带着痞气但靠谱
- 读《活着》时，AI 化身为**福贵**——用沧桑的口吻讲述一生
- 读《西游记》时，AI 化身为**孙行者**——机灵狡黠，火眼金睛，一起讨论哪只妖怪战斗力最高
- *更多书籍持续加载中……*

### 💬 陪伴式引导
- **每晚提醒**：定时发送个性化消息，询问阅读进度，触发阅读动作
- **主动提问**：读到关键情节时，AI 主动问“想听听福贵怎么说的吗？”
- **随时讨论**：阅读中随时发送感想，AI 即时回应，引用原文、多角度分析

### 📝 自动整理笔记
- 每日发送一段文字感想，AI **自动提取金句**、关联过往阅读
- 一本书读完后，AI 汇总生成 **完整结构化读书笔记**，可直接分享到社交平台

### 🧭 智能推荐书籍
- 基于对话内容推荐下一本书：“上次你和叶文洁聊人性，也许可以和拉斯柯尼科夫聊聊”

- 生成个人 **“思想地图”**，可视化展示阅读轨迹与思想演变

## ✨ 效果如何

- **阅读量提升**：沉浸式的体验，让用户沉浸式的和人物对话
- **形成闭环**：“想读 → 在读 → 读完 → 想读”的可持续阅读循环
- **缓解孤独**：让独居的人、深夜的人、独自面对一本书的人，都有了可以对话的对象

## 🧠 技术亮点

基于 OpenClaw 框架 技术：

- **多层级知识库**：全书文本  + 对话历史等
- **语言风格模仿**：精确还原人物的说话方式、思维模式、价值观念
- **持久记忆**：记住用户与每个人物的每一次对话，延续交流
- **本地优先**：阅读记录、对话内容本地存储，保护隐私
